# Demo Scenarios

The first version includes 3 built-in demo scenarios so users can try the skill immediately.

## marketing-site

- goal: quickly demonstrate recommendation and build validation from a document to a landing-site starter
- default capabilities: optional `edge-config` only
- default starter: `nextjs-marketing-starter`

## saas-mvp

- goal: demonstrate recommendation and validation for the default `clerk + neon` combination
- default capabilities: `clerk`, `neon`, optional `edge-config`
- default starter: `nextjs-saas-starter`

## upload-app

- goal: demonstrate how upload-oriented apps default to `blob`
- default capabilities: `blob`
- default starter: `nextjs-blob-upload-starter`
